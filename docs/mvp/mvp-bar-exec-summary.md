# Executive Summary: Проект "Бар самообслуживания" (MVP v1.0)

**Дата:** 14 октября 2025 г.

### **1. Суть проекта**

**Beer Tap System** — это завершенный MVP (минимально жизнеспособный продукт) аппаратно-программного комплекса для автоматизации бара. Система позволяет гостям самостоятельно наливать напитки с помощью RFID-карт, при этом обеспечивая точный учет финансов и остатков.

### **2. Ключевые возможности (Что сделано)**

-   **Автономная работа:** Точки налива (контроллеры Raspberry Pi) продолжают работать и фиксировать продажи даже при потере связи с сервером.
-   **Централизованное управление:** Разработано десктопное приложение (**Admin App**) для персонала, позволяющее управлять гостями, их балансом, RFID-картами, а также инвентарем (напитками, кегами, кранами).
-   **Транзакционная надежность:** Все финансовые и инвентарные операции (пополнения, наливы) являются транзакционными, что гарантирует целостность данных.
-   **Полный аудит:** Система автоматически ведет журнал всех ключевых административных действий, обеспечивая полную прозрачность.
-   **Готовность к развертыванию:** Вся серверная часть упакована в Docker и разворачивается одной командой.

### **3. Технологический стек**

-   **Сервер (Backend):** FastAPI (Python), PostgreSQL.
-   **Приложение для персонала (Admin App):** Tauri (Svelte + Rust).
-   **Контроллер крана (IoT):** Python, SQLite на Raspberry Pi.

### **4. Статус проекта**

Проект достиг стадии **"MVP Complete"**. Все запланированные на первом этапе функции реализованы, протестированы (включая автоматические тесты) и задокументированы. Система готова к пилотному внедрению.

### **5. Следующие шаги (План развития)**

-   Разработка модуля расширенной отчетности и аналитики.
-   Внедрение функционала управления сменами.
-   Интеграция с внешними платежными системами.
